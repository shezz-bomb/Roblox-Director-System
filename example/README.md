# Example Place

Para probar el sistema rápidamente:

1. Abre Roblox Studio y crea un nuevo lugar vacío.
2. Copia las carpetas `DirectorSystem` y `Dependencies` dentro de `ServerScriptService`.
3. Crea un Script en `ServerScriptService` con:
   `require(game.ServerScriptService.DirectorSystem.DirectorController):start()`
4. Ejecuta el juego. El Director comenzará a lanzar eventos.

*No se incluye un archivo .rbxl para mantener el repositorio ligero.*
